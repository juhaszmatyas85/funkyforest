Flex Guten is a multipurpose Gutenberg block plugins. This has all necessary Gutenberg blocks like Pinterest Pinit Block, post grid block, testimonial block blocks.

== Description ==

Flex Guten allow you to add stunning design blocks on your Gutenberg website. We have built several blocks that you can use for free. We are working on more blocks that we will release here soon.

Additionally, you can request new features though our support page. We will add your requested feature in next update. We are building more blocks using our check lists.

If you find any bug on our plugins, you can request support through our support page.

== Installation ==
* Download the plugins from WP Directory Page.
* Install the plugins on your website
* Activate it & enjoy the features.
